# DNF 台服服务端一键安装包

## 说明

参考[XanderYe/dnf](https://github.com/XanderYe/dnf)

本包基于 CentOS 7 进行测试，其他系统未测试，可能存在部分问题。
使用一键包前，请先确认您的系统为干净的原始系统，并保证系统为最小化安装。

## 安装

```shell
chmod +x install.sh
./install.sh
```

安装完成后会自动启动游戏服务端，视服务器情况，具体出现五国时间不定。
